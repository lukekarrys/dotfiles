# lukekarrys/dotfiles

## Prereqs

- [`brew`](https://brew.sh)
    - `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
- [`chezmoi`](https://www.chezmoi.io/install/#one-line-package-install)
    - `sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply lukekarrys`
